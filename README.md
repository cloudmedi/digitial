# Maia Signage
Maia Signage Rest/WS API Framework with moleculer.services

## Installation

``$ npm pm2 --global``

``$ git clone git@github.com:cloudmedi/signage_api.git``

``cd signage_api && npm i``

``pm2 start ecosystem.config.js``
